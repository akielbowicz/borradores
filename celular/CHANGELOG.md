# Termux Setup Changes

## 2025-11-16

### Added
- Nushell, Zoxide, Yazi, Starship, Ripgrep, Bat, Helix, Zk, Carapace, GitHub CLI
- All tools configured with monochrome high-contrast themes (black/white only)
- Git aliases optimized for two-thumb Colemak DH typing
- Setup scripts in `~/dev/borradores/celular/termux-setup/`

### Git Aliases (Thumb-Optimized)
- `gn` - git status
- `ge` - git add .
- `gt` - git commit
- `gh` - git push
- `gd` - git diff
- `gr` - git pull
- `gi` - git commit -m

See `GIT_ALIASES_COLEMAK.md` for details.

### Skipped
- UV (Python) - not available for Android ARM
- Juliaup (Julia) - not available for Android ARM
